# 🛡️ Cybersecurity Internship Portfolio
**Author:** Mohammed Fadil PK
**Domain:** Cybersecurity & Digital Forensics

Welcome to my cybersecurity internship portfolio! This repository contains all the automated tools and security scripts I developed during my internship program.

## 📂 Projects Included:

### 1. [Project 4: Password Strength Analyzer & Wordlist Generator](./Project%204)
A Red Team offensive security toolkit. It calculates password entropy and utilizes OSINT data to algorithmically generate customized dictionary wordlists (with leetspeak permutations) for brute-force attacks.

### 2. [Project 8: Linux Hardening Audit Tool](./Project%208)
An automated defensive script built to verify Linux system security configurations. It programmatically checks UFW firewall status, SSH root login settings, and network IP forwarding rules.

### 3. [Project 11: Log File Analyzer for Intrusion Detection](./Project%2011)
A Blue Team Intrusion Detection System (IDS). This tool parses system authorization logs using Regex to detect SSH brute-force attacks and automatically generates a visual threat dashboard using `matplotlib`.

### 4. [Project 13: Steganography Data Concealment Tool](./Project%2013)
A custom GUI desktop application built for digital forensics. It utilizes the Least Significant Bit (LSB) method to secretly embed and extract hidden text payloads within the microscopic pixel data of standard image files.
